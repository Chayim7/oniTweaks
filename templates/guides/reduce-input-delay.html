What Is Input Delay?

Input delay is the time between when you press a button and when that action occurs in-game. In competitive gaming, even small delays (10-50ms) noticeably impact performance.

Input delay comes from multiple sources: your controller/mouse, hardware processing, game engine, network connection, and display. Understanding each factor is critical for optimization.

Why Input Delay Matters in Fortnite

Fortnite is a fast-paced game where split-second reactions determine outcomes. Input delay affects:

  • Aiming: Delayed input makes tracking and flicking feel sluggish
  • Building: Delayed responses slow down build speed and edit execution
  • Combat Reaction: Slower response to enemy actions puts you at disadvantage
  • Consistency: Variable delay creates unpredictable gameplay feel

Sources of Input Delay

Input delay accumulates from multiple sources. Understanding each helps identify optimization opportunities:

Controller Delay

Modern controllers vary in input latency (2-10ms). Wired controllers typically have less delay than wireless.

System Processing Delay

Your PC/Console processes input and renders frames. Higher performance reduces this delay.

Game Engine Delay

Fortnite's engine processes input at specific intervals. Higher FPS can reduce perceived delay.

Network Latency (Ping)

Your connection to the server adds delay. Most players see 20-150ms ping. Even good connections add noticeable lag.

Display Latency

Monitor response time (pixel change speed) adds delay. Gaming monitors (1-5ms) are faster than typical displays (20-50ms).

What Actually Reduces Input Delay

Here are factors that genuinely impact input delay in Fortnite:

  • Higher FPS: More frames = less time between input and response
  • Lower Ping: Closer servers and optimized connection reduce network latency
  • Gaming Monitor: Lower response time reduces display delay
  • Wired Connection: Ethernet reduces network variance vs WiFi
  • USB Controller: Wired connection slightly reduces controller latency

Common Misconceptions

Many things are blamed for input delay but don't actually affect it much:

  • Game Settings: Graphics settings don't directly reduce input delay (though higher FPS does)
  • Controller Sensitivity: High or low sensitivity doesn't create delay
  • V-Sync On/Off: Modern V-Sync implementations don't add much delay

Why Incomplete Optimization Fails

Focusing on only one factor doesn't fully reduce input delay. Effective optimization addresses:

  • Hardware-level latency (monitor, controller)
  • System performance (FPS consistency)
  • Network optimization (ping stability)
  • Game settings (configured for responsiveness)
